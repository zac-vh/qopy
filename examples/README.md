# Examples

This folder contains both research notebooks and minimal runnable scripts.

## Runnable scripts

- `examples/quickstart_wigner.py`
  - Builds a few Wigner functions and saves an illustration to `docs/assets/wigner_gallery.png`.
- `examples/quickstart_state_space.py`
  - Demonstrates ket -> density -> channel -> measures workflow.

Run them with:

```bash
PYTHONPATH=. python3 examples/quickstart_wigner.py
PYTHONPATH=. python3 examples/quickstart_state_space.py
```
